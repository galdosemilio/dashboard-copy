1. Inactive Clinics Report cloud_download

    All clinics over the last 30 days, with a listing of the 'active' coaches and patients within that clinic over that time period. 'Active' is defined as any API activity during the last 30 days. The CSV report has the following columns:

    • Organization ID The ID of the organization being reported
    • Organization Name The name of the organization being reported
    • Organization Hierarchy The list of organization IDs that precede the one being reported (inclusive)
    • Providers Total The total number of coach accounts within this organization
    • Providers Active Count The number of coach accounts within this organization that had any activity over the last 30 days
    • Providers Active Percentage The percentage of coach accounts within this organization that had any activity over the last 30 days
    • Clients Total The total number of patient accounts within this organization
    • Clients Active Count The number of patient accounts within this organization that had any activity over the last 30 days
    • Clients Active Percentage The percentage of patient accounts within this organization that had any activity over the last 30 days
