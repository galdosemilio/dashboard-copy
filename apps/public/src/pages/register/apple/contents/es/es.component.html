Cómo configurar tu cuenta

Para alojar tu aplicación iOS (iPhone y iPad) en la tienda de aplicaciones de Apple, necesitas una Cuenta de Desarrollador de Apple. Este documento te guiará en la inscripción del Programa de Desarrolladores de Apple.

Después de configurar tu Cuenta de Desarrollador de Apple (o en caso de que ya tengas una), invita a CoachCare a colaborar en tu cuenta. CoachCare te ayudará a administrar tu aplicación y la mantendrá actualizada y funcionando correctamente.

El proceso completo de registro de tu Cuenta de Desarrollador de Apple tomará entre unos pocos días y unas pocas semanas. Para ayudar a agilizarlo, asegúrate de enviar todos los materiales correctos y responder rápidamente a cualquier comunicación de Apple.

Antes de empezar

Si ya tienes una Cuenta de Desarrollador de Apple para tu empresa, puedes proceder directamente a la sección de Invitar a CoachCare a tu Cuenta de Desarrollador.

Para crear una cuenta para tu empresa, necesitarás lo siguiente:

  1. Una computadora o dispositivo en el que se pueda utilizar la autenticación de dos factores de Apple. Puede ser un dispositivo iOS (iPhone, iPad o iPod Touch), una computadora de escritorio o un portátil con sistema Apple, o también una computadora de escritorio o portátil con sistema Windows. Si tienes alguna pregunta sobre este requisito, comunícate con nuestro Equipo de Implementación.
  2. Organización legal o nombre de la empresa
  3. Un sitio web disponible públicamente con un nombre de dominio asociado a tu empresa.
  4. Un número de teléfono de contacto al que Apple llamará para verificar la información de tu empresa.
  5. Una dirección de correo electrónico, asociada con tu empresa, que se utilizará para registrar la cuenta.

Proceso de inscripción para desarrolladores de Apple

  1. Crea una ID de Apple para tu empresa

    Lo primero que debes hacer es crear una ID de Apple para tu empresa, que luego utilizarás para crear una Cuenta de Desarrollador de Apple para tu empresa. No es posible utilizar tu propia ID de Apple personal. Además, la dirección de correo electrónico utilizada para tu ID de Apple debe tener el mismo nombre de dominio del sitio web de tu empresa. No puedes utilizar una cuenta de correo electrónico con dominio general (@ gmail.com, @ yahoo.com, @ hotmail.com, etc.). Si aún no tienes un ID de Apple para tu empresa, debes crear un ID de Apple.

  2. Configura la autenticación en dos pasos para tu ID de Apple

    Desde el 29 de junio de 2018, Apple introdujo el requerimiento de que las cuentas empresariales de ID de Apple cuenten con una autenticación de dos factores para mejorar la seguridad de las mismas. Debes habilitar la autenticación de dos factores para tu cuenta de ID de Apple. Ten en cuenta que este paso requiere de un dispositivo Apple o una PC de Windows con iCloud para Windows e iTunes instalados. Considera que debes iniciar sesión con el ID de Apple de la empresa en el dispositivo o la computadora que se uses para la autenticación de dos factores. Hay dos opciones disponibles:

    1. Enfoque Recomendado Compra y/o mantén un dispositivo Apple (iPhone, iPad, computadora, computadora portátil) o computadora con Windows que esté siempre conectado con este ID de Apple. Este dispositivo o computadora solamente podrá ser utilizado para la empresa y no podrá ser usado como tu dispositivo o teléfono personal.
    2. Usar un dispositivo o teléfono personal e iniciar sesión con la ID de Apple de tu negocio, cuando sea necesario. Por ejemplo:
      • Tu iPhone personal normalmente está conectado a tu cuenta personal de ID de Apple.
      • Cuando sea necesario para las actividades de negocios, cierra sesión en tu cuenta personal de ID de Apple en tu teléfono o computadora, e inicia sesión con tu ID de Apple empresarial.
      • Después de que la autenticación se complete (debería tomar unos pocos minutos), cierra sesión en la cuenta de ID de Apple de tu empresa e inicia sesión en tu cuenta personal de ID de Apple.
      En general, las clínicas necesitarán acceder a sus cuentas comerciales de Apple ID una vez cada 2-3 meses, por lo que esta práctica es posible de implementar sin problemas.
  3. Registrar o localizar el número D-U-N-S

    Tu organización debe tener un Número D-U-N-S para que Apple pueda verificar la identidad de la misma y el estatus de la entidad legal. Dun & Bradstreet asigna estos números únicos de nueve dígitos y se utilizan en general como identificadores comerciales estándar.

    • Regístrate o localiza tu número D-U-N-S
      Se requiere una ID de Apple antes de registrarse para obtener un número D-U-N-S.
      Por favor, completa el paso anterior (crear ID de Apple) antes de continuar con este punto.
    • Más información sobre el número D-U-N-S.
  4. Registrarse para una Cuenta de Desarrollador de Apple

    A continuación, completa el formulario de registro para inscribirte en el programa Cuenta de Desarrollador de Apple.

    Al inicio del proceso de registro, regístrate como una Empresa/Organización. Si tienes alguna pregunta sobre tu tipo de entidad o prefieres registrarte como un tipo diferente, comunícate con nuestro Equipo de Implementación antes de continuar.

    Después de completar este proceso, un representante de Apple se pondrá en contacto contigo a la dirección de correo electrónico y el número de teléfono que hayas proporcionado dentro de las 24 a 72 horas posteriores. Apple verificará la información de tu empresa.

  5. Introducir información sobre el pago

    Una vez que Apple haya verificado la cuenta, recibirás un correo electrónico de confirmación. En el mismo, se incluirán instrucciones para pagar la membresía del Programa de Desarrolladores de Apple.

Invitar CoachCare a tu Cuenta de Desarrollador de Apple

Una vez que tu Cuenta de Desarrollador de Apple esté completamente activa, deberás invitar a CoachCare como Administrador a tu cuenta de App Store Connect. Para empezar:

  1. Dirígete a la sección de Usuarios y Acceso en tu cuenta de App Store Connect.
  2. Haz clic en la pestaña de Personas en la parte superior de la página.
  3. Haz clic en el botón "+" para añadir un nuevo usuario (Coachcare).
  4. Utiliza "CoachCare" para los campos de Nombre y Apellido.
  5. Para el "correo electrónico", utiliza mobile@coachcare.com.
  6. En Roles activa la casilla de Administrador.
  7. Luego, haz clic en el botón de Invitar en la parte inferior.

El proceso debería estar completo. Puedes solicitar asistencia o dirigir tus preguntas a la página de Soporte de CoachCare o a tu ejecutivo de ventas regional.
